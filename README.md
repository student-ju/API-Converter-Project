# 💱 Currency Converter Application

A sleek, responsive web application that provides real-time currency exchange rates. This project demonstrates the practical application of fetching external data through APIs and dynamically updating the Document Object Model (DOM).

## 🚀 Live Demo
[View the Live Project here](https://student-ju.github.io/API-Converter-Project/)

## ✨ Features
- **Real-time Conversion:** Fetches the latest exchange rates using a Currency API.
- **Dynamic Flag Integration:** Automatically updates country flags based on the selected currency using `flagsapi.com`.
- **Responsive UI:** Clean, mobile-friendly design for seamless use across devices.
- **Error Handling:** Robust input validation to ensure accurate conversions.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling and responsive layouts (Flexbox).
- **JavaScript (ES6+):** Fetch API for asynchronous data, Event Listeners, and DOM Manipulation.

## 📖 How to Use
1. Enter the amount you wish to convert.
2. Select the "From" and "To" currencies using the dropdown menus.
3. Click **Get Exchange Rate** to view the current value.

## 📂 Project Structure
- `index.html` - The core structure of the app.
- `style.css` - Custom styling and animations.
- `script.js` - Logic for API calls and UI updates.
- `codes.js` - Data mapping for currency codes and country flags.
